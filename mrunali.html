Project Title :Maintaining a traffic control at rush places using IOT

Objective : Handling a traffic issue at rush places in metro cities by using smart systems

Problem statement : In metro cities, traffic issue is a major issue. So this issue can be handled by using sensors.

Solution : Now we are considering that we want to go to a movie or drama shows in our city. Those places are full of rush. The user will check for ticket availability for the show he want. We will create a website in which we are feeding data as area of that parking. Also, we are inserting a one sensor which will keep track of how many vehicles are entering in the parking. Along with area of parking we are feeding area required for one vehicle. (This calculations we are doing by consideration of two wheeler parking) Then as vehicles will enter the parking count will be incremented and if vehicle leaves parking then count is decremented. Before going to the place that user wants he will check first at our website and will check is the space available. If space is available we will display a message that “yes you can go “,otherwise it will display “no you have to wait”. In this we are dealing with one parking area which we select and we will insert sensor at that place.
